# Font Subset Web

一个基于浏览器本地运行的字体子集化工具：上传字体文件，选择字符集，生成更小的 `woff2` 文件并提供接入代码。
🤚在线地址：https://Nicander93.github.io/font-subset-tool/
## 你可以做什么

- 支持上传 `ttf/otf` 源字体
- 支持预设字符集（常用 3500 / 3500+英数标点）
- 支持手动追加字符和上传字符集文本
- 浏览器本地 wasm 处理，不上传字体到服务端
- 输出 `woff2`，展示压缩体积和压缩率

## 如何使用

1. 上传字体文件（`.ttf` 或 `.otf`）
2. 选择字符集方案（常用 3500 或 3500+英数标点）
3. 如有需要，追加自定义字符或上传 `.txt` 字符集
4. 点击「开始子集化」
5. 下载生成的 `woff2` 文件并复制接入 CSS

## 输出结果

- 生成文件：`*.subset.woff2`
- 页面会显示原始体积、输出体积、压缩率、处理耗时
- 页面会提供 `@font-face` 代码片段可直接复制

## 隐私与安全

- 全程浏览器本地处理，不上传字体文件到服务端
- 刷新页面后，当前处理结果不会保留

## 使用建议

- 建议优先使用业务真实文案提取字符集，避免缺字
- 若页面仍出现缺字，请补充字符后重新子集化
- 对于超大字体文件，建议先确认仅保留必要字重和字符

## 当前限制

- 输入格式仅支持 `.ttf/.otf`
- 输出格式为 `.woff2`
- 首版以常见中文网页场景为主
